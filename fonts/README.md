# Cover typography fonts

Open-license display fonts bundled for app-side cover typography
(`server/lib/coverTypography.js`, rendered via `@resvg/resvg-js`).

All fonts are redistributable and embeddable in web + print. Sourced from the
[google/fonts](https://github.com/google/fonts) repository.
