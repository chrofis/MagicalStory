# AI Story Creator

A multilingual AI-powered children's story generator that creates personalized stories based on custom characters and relationships.

## Features

- **Multi-language support**: English, German, and French
- **Character creation** with detailed attributes (strengths, weaknesses, fears)
- **Relationship mapping** between characters
- **Story customization**: Choose story type, page count, and reading level
- **AI-powered story generation** using Claude AI
- **Export/Import** configurations as JSON
- **Download stories** as TXT or PDF

## Setup Instructions

### For GitHub Deployment

1. Get your Anthropic API key from [console.anthropic.com](https://console.anthropic.com/)
2. Upload this repository to GitHub
3. Enable GitHub Pages in repository settings
4. Access your story creator at `https://yourusername.github.io/MagicalStory/`

### For IONOS Hosting

1. Upload the `index.html` file to your IONOS web hosting
2. Access it via your domain
3. Enter your Anthropic API key when prompted (it's stored securely in your browser)

## Using the Application

1. **Select Language**: Choose English, German, or French
2. **Choose Story Type**: Pick from 10 different adventure themes
3. **Create Characters**: Add characters with photos and personality traits
4. **Define Relationships**: Set how characters relate to each other
5. **Configure Story**: Select main characters, pages, and reading level
6. **Generate**: Click to create your AI-generated story!

## API Key

This application requires an Anthropic API key to generate stories. Your API key is:
- Stored locally in your browser
- Never sent anywhere except directly to Anthropic's API
- Can be updated anytime in the application
