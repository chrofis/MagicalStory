6. Referral economics

Configured in server/config/credits.js → REFERRAL:

  • Buyer gets CHF 10 off the printed book at checkout.
  • Referrer gets 350 credits (= CHF 10 starter pack) granted on confirmed payment.
  • One referral code per buyer, ever (users.referred_by). No self-referral. Code format: MagicName{NNN} with 900 slots per first-name.

Endpoints in server/routes/print.js: GET /api/referral/my-code, POST /api/referral/validate. Webhook in server.js credits the referrer transactionally with an atomic referred_by lock.
