Detect ALL figures, faces, and notable objects in this storybook illustration.

Return bounding boxes for EVERY element in the image:

COORDINATE SYSTEM:
- Values are 0-1000 (Gemini native format)
- Origin is top-left corner
- Format: [y_min, x_min, y_max, x_max]

FOR EACH FIGURE (character/person):
- face_box: bounding box around the FACE only (forehead to chin, ear to ear)
- body_box: bounding box around the FULL BODY (head to feet, or as much as visible)
- If face is not visible (back view), set face_box to null

FOR EACH NOTABLE OBJECT:
- body_box: bounding box around the object
- face_box: null
- Include: held items, vehicles, animals, furniture, magical elements

Return ONLY valid JSON with no other text:
{
  "figures": [
    {
      "label": "visual description (e.g., 'boy in blue hoodie on left')",
      "position": "left|center|right",
      "face_box": [y_min, x_min, y_max, x_max],
      "body_box": [y_min, x_min, y_max, x_max]
    }
  ],
  "objects": [
    {
      "label": "visual description (e.g., 'red bicycle in foreground')",
      "position": "left|center|right",
      "body_box": [y_min, x_min, y_max, x_max]
    }
  ]
}
