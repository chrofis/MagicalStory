COORDINATE SYSTEM:
- Values are 0-1000 (Gemini native format)
- Origin is top-left corner
- Format: [y_min, x_min, y_max, x_max]

FOR EACH FIGURE:
- If you can identify it as an expected character, set "name" to the character name (EXACTLY as listed above)
- Use CLOTHING + POSITION to distinguish characters (two boys of similar age are distinguished by what they wear and where they stand)
- If not identifiable as any expected character, set "name" to "UNKNOWN"
- Include "confidence": "high", "medium", or "low" for identification certainty
- face_box: bounding box around the COMPLETE FACE including all hair.
  Must include: top of hair/forehead to bottom of chin, and full width from ear to ear including any hair that extends outward.
  Common mistakes to AVOID:
  * Do NOT cut off the top of the head or hair — include ALL hair above the forehead
  * Do NOT cut off the chin — extend below the jawline
  * Do NOT make the box too narrow — include ears and hair width
  * When in doubt, make the face_box LARGER rather than smaller
  * The face_box should be roughly square for a front-facing character
- body_box: bounding box around the FULL BODY (or as much as visible)
- If face is not visible (back view, non-human creature), set face_box to null

FOR EXPECTED OBJECTS:
- Report "found": true/false for each expected object
- If found, include body_box bounding box and position

Return ONLY valid JSON:
